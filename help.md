# Running SupplierMate

1. If ruby is not installed on your computer, follow the link below.

Link: <https://www.ruby-lang.org/en/documentation/installation/>

2. Install gem bundler. 

- Navigate to the SupplyMate file in your terminal.
- Run - gem install bundler.

3. Clone this repository to your local computer.

4. Navigate to the src folder in the repository. 

5. Two options when running the app. 
If you want a personalised welcome page, run: ruby main.rb [any name]

If you are happy to run the app with no personalisation, run: ruby main.rb
